== Description ==
his plugin Enables customers to add extra amount on checkout.
They have the ability on checkout to input any extra amount, with fancy increase and decrease buttons. The Fee would be added alongside the regular order on order summary and also shows up on the email receipt.

This plugin is great for giving delivery personnel Tips! You can even configure it your self to suite your needs.


== Installation ==

1. Upload the entire 'add-tip-woocommerce' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy! ;)
